# SHIBUMI · Intake — הוראות הפעלה

תסריט Google Apps Script שבונה מערכת איסוף מידע מלאה: טופס Google Forms, גיליון Google Sheets עם 66 הנכסים הפעילים, תיקיית Drive לתמונות, וגיליון תגובות — הכל תחת תיקייה אחת ב-Drive.

---

## הפעלה (פעם אחת, 2 דקות)

1. פתח דפדפן ולך ל: **<https://script.google.com>**
2. לחץ **New project** (או: פרויקט חדש)
3. מחק את התוכן שבעורך (`function myFunction() {}`)
4. פתח את הקובץ **`SHIBUMI_intake.gs`** (בתיקייה הזו)
5. העתק את **כל** תוכנו → הדבק בעורך של Apps Script
6. שמור (Ctrl+S) — תן לפרויקט שם, למשל "SHIBUMI Intake"
7. בפינה העליונה ליד כפתור Run, בחר את הפונקציה **`setupShibumiIntake`** מה-dropdown
8. לחץ **Run** (▷)
9. בפעם הראשונה יופיע חלון **Authorization required**:
   - לחץ **Review permissions**
   - בחר את חשבון ה-Google שלך
   - לחץ **Advanced** → **Go to SHIBUMI Intake (unsafe)** (זה בגלל שזה פרויקט אישי לא-מאומת)
   - אשר את ההרשאות (Drive, Forms, Gmail, Sheets)
10. חזור ל-Apps Script ולחץ Run שוב. זה לוקח ~20 שניות.
11. בתחתית — ב-**Execution log** — יופיעו כל הקישורים:
    - 📁 תיקיית Drive
    - 📝 קישור לטופס (לשליחה ל-Michael)
    - 📊 גיליון נכסים למילוי
    - 📷 תיקיית תמונות
    - 📥 גיליון תגובות

(בנוסף — כל הקישורים יישלחו גם במייל לחשבון שלך אוטומטית.)

---

## מה Michael מקבל

שלח לו שני קישורים:

1. **הטופס** (Google Forms) — ~15 דקות, עם הסתעפויות חכמות
2. **תיקיית התמונות** (Google Drive) — שם יעלה תמונות/סרטונים

הטופס מתחיל בשאלות על החברה (טלפון, אודות, צוות, המלצות, רשתות חברתיות), ואז מסתעף:
- "**יש לך קובץ עם פרטי הנכסים?**" → אם כן, לינק ל-Drive. אם לא, לינק לגיליון שהכנו עם 66 נכסים.
- "**יש לך תמונות איכותיות?**" → אם כן, איך לשלוח (Drive / ZIP / WhatsApp). אם לא, מה לעשות.

## שינויים / הרצה חוזרת

- אם תריץ **שוב** את `setupShibumiIntake()` — הוא ימצא את התיקייה הקיימת וישתמש בה. הטופס והגיליונות **יווצרו מחדש** (לא ימחקו ישנים — רק יופיעו כעותקים חדשים). אם אתה לא רוצה כפילויות — מחק את הישנים מה-Drive לפני ההרצה.
- כדי לשנות שאלות — ערוך את הפונקציה `createIntakeForm_` בתוך הקובץ, ואז הרץ שוב.
- הנתונים של הנכסים (`PROPERTY_ROWS`) נבנים מתוך `shibumi_properties/properties.json` — אם תעדכן את הסקרייפר, תוכל לחדש את הקובץ.

## שאלות נפוצות

**Q: למה 66 ולא 70?**  
A: 4 נכסים מסומנים כ"נמכר" / "הושכר" / "לא אקטואלי" והוחרגו אוטומטית.

**Q: אפשר לעשות את זה בלי Google Forms?**  
A: כן — הגיליון לבדו יכול להיות מקור מידע עצמאי. אבל Michael יקבל טופס יפה וקל יותר.

**Q: התמונות יעברו ישר לאתר?**  
A: לא. אחרי ש-Michael יעלה אותן — נוריד אותן, נעבד, ונכניס ל-`shibumi_properties/images/` ואז לאתר.
